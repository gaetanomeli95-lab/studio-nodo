# Studio Nodo

Sito web premium per Studio Nodo, personal brand freelance di Gaetano Meli a Palermo.

## Stack

- Next.js
- TypeScript
- Tailwind CSS
- Framer Motion

## Comandi

```bash
npm install
npm run dev
npm run build
npm run start
```
